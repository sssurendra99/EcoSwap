# Ecoswap - A Sustainable marketplace

## Getting Started

1. Clone the project

```bash
https://github.com/sssurendra99/EcoSwap.git
```

2. Run the project with maven.

```bash
./mvnw spring-boot:run # on Linux or mac
.\mvnw spring-boot:run # on Windows
```
